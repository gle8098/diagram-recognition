Сборка для MacOS x86_64.

FAQ.

Q: Что делать, если вместо приложения открывается окно
 "Файл «godr.app» невозможно открыть, так как Apple не
  может проверить его на наличие вредоносного ПО."

A:
1. Запустить godr.app, увидеть сообщение и нажать Cancel.
2. Открыть System Preferences -> Security & Privacy,
   или «Защита и безопасность» в системных настройках.
3. Поставить внизу галочку Allow Anyway, или «Любого источника»,
   напротив названия программы.
4. Запустить godr.app, снова увидеть окно и нажать Open.

Скриншоты можно найти по ссылке ниже для англоязычной MacOS.
https://github.com/hashicorp/terraform/issues/23033#issuecomment-542302933
